Features

This component has the following features:
  • Reactive
  • Lazy loading
  • 1 to 1 mapping
  • template keys

Reactive

Everytime a data-i18n-* attribute is added or changed inside the component, its respective content will be automatically updated to reflect the content

Lazy loading

Only the necessary content is loaded to apply the internationalization
